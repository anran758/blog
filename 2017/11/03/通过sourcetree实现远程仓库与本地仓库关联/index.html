通过sourcetree实现远程仓库与本地仓库关联

发表于 2017-11-03 | 分类于 Git |

  sourceTree 是当下比较流行的git客户端,简化了如何与Git存储库进行交互，从而可以专注于编码, 可以通过简单的Git GUI可视化来管理仓库, 提高效率.

  题归正转, 今天我们来说说如何通过sourcetree来将本地仓库与远程仓库关联.

步骤

 首先把本地仓库初始化.

 再添加到sourcetree上

 进入github.com -> your profile -> repositories(仓库) -> New

 填写仓库信息, 复制仓库链接

 设置 -> 添加 -> 填写信息 -> 确定

 然后就可以提交到github了~
